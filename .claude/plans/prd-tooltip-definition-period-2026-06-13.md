# PRD: 各指標の定義・期間ツールチップ機能

作成日: 2026-06-13

## 1. 概要

銘柄分析画面の各指標ラベル横に「？」アイコンを配置し、クリックするとその指標の定義と分析に使ったデータ期間をツールチップで表示する。投資家が各数値の意味と根拠期間を即座に把握できるようにする。

## 2. 目的・成功指標

- 目的: 各財務指標・スコアの意味と使用データ期間を1クリックで確認できるようにする
- 成功の定義: 全対象指標に「？」アイコンが表示され、クリック時に定義と期間が正しく表示される

## 3. スコープ

### In Scope（やること）
- AI総合スコアセクションの各スコア項目への「？」追加
- 下落理由分析セクションの各判定項目への「？」追加
- 財務分析セクションの各指標（ROE、ROA、売上成長率など）への「？」追加
- 定義テキスト（静的・コード固定）の記述
- 期間の動的表示（APIデータから取得）

### Out of Scope（やらないこと）
- 投資スタイル判定セクションへのツールチップ追加
- 定義テキストの管理画面・外部編集機能
- ホバーでの自動表示（クリックのみ）
- 英語対応

## 4. ユーザーストーリー

- 投資家として「ROE」の意味を知りたい（理由：指標名だけでは意味がわからないため）
- 投資家として「売上成長率がどの期間のデータか」を知りたい（理由：古いデータとの比較を避けるため）
- 投資家として「AI総合スコアは何のデータを元に算出されたか」を知りたい（理由：スコアの信頼性を判断するため）

## 5. 機能要件

### Must Have（必須）
- 各指標ラベル横に「？」アイコン（小・グレー）を表示
- クリックでツールチップを表示・再クリックまたは外側クリックで閉じる
- ツールチップには「定義」と「期間」を1つにまとめて表示
- 定義テキストはコードに静的記述
- 期間は以下のルールで動的表示：
  - 財務分析系: `CurPerSt〜CurPerEn`（例：2024/4/1〜2025/3/31）
  - AI総合スコア: 財務データと同じ期間
  - 下落理由分析: 株価データの実際の日付範囲（例：2025/5/14〜2026/6/13）

### Should Have（推奨）
- モバイルでも操作しやすいタップ領域（最小44×44px）
- 同時に複数のツールチップが開かない（1つのみ）

### Won't Have（今回はやらない）
- ツールチップ内のリンク・動的コンテンツ
- 定義テキストのi18n対応

## 6. データモデル

### 定義テキスト（静的・コード内オブジェクト）

```ts
type MetricDefinition = {
  label: string       // 指標名
  definition: string  // 定義説明（2〜3文）
  periodType: 'financial' | 'stockPrice' | 'none'
}
```

### 期間データ（APIから受け取る）

```ts
type PeriodInfo = {
  financialPeriod: {
    start: string  // CurPerSt (例: "2024-04-01")
    end: string    // CurPerEn (例: "2025-03-31")
  } | null
  stockPricePeriod: {
    start: string  // 株価データ最古日付
    end: string    // 株価データ最新日付
  } | null
}
```

## 7. API設計

既存の `/api/stock/[code]` レスポンスに以下を追加する：

```ts
// analysis オブジェクトに追加
periodInfo: {
  financialPeriod: { start: string; end: string } | null
  stockPricePeriod: { start: string; end: string } | null
}
```

`CurPerSt` / `CurPerEn` は `fins/summary` の通期レコードから取得済み。これを `parseStatements` から返す形に拡張する。

## 8. UI/UX 概要

### ツールチップコンポーネント（共通）
```
[指標名] [?]
           ↓クリック
    ┌─────────────────────┐
    │ 📌 定義              │
    │ ROEとは、株主資本に対 │
    │ する当期純利益の比率  │
    │ です。               │
    │                     │
    │ 📅 データ期間        │
    │ 2024/4/1〜2025/3/31 │
    └─────────────────────┘
```

### 配置対象（全指標）
**財務分析**
- 売上成長率、営業利益成長率、EPS成長率
- 自己資本比率、営業CF、有利子負債比率
- ROE、ROA、営業利益率
- 配当利回り、配当性向
- PER、PBR

**AI総合スコア**
- 財務健全性スコア、成長性スコア、収益性スコア、株主還元スコア、割安性スコア、テクニカルスコア

**下落理由分析**
- 市場要因、銘柄固有要因、各判定項目

## 9. 非機能要件

- パフォーマンス: ツールチップ表示はクライアントサイドのみ（追加API不要）
- セキュリティ: 定義テキストは静的のためリスクなし
- モバイル対応: タップで開閉、外側タップで閉じる

## 10. エッジケース・エラー処理

- `CurPerSt` / `CurPerEn` が取得できない場合: 期間欄を「取得できませんでした」と表示
- 株価データが0件の場合: 期間欄を非表示
- ツールチップが画面端にはみ出る場合: 左右反転表示

## 11. テスト方針

- 各セクションの「？」アイコンをクリックして定義が表示されること
- 期間が正しいフォーマット（YYYY/MM/DD〜YYYY/MM/DD）で表示されること
- 外側クリックでツールチップが閉じること
- モバイルサイズでタップ可能なこと
